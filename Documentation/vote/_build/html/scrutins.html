confrontation2(urne)¶

Rassemble une liste de bulletins de vote dans une matrice de préférences. les “bulletins” devraient être une liste de bulletin, où chaque bulletin est une liste de classements. Pour par exemple, s’il y a 3 candidats, chaque bulletin de vote doit être une liste de 3 numéros correspondant aux candidats. Les classements les plus petits sont

meilleurs.
